# v0.6.14

This patch release adds passwordless self-update support and exposes update checks in Feishu.

## Changes

- Added `ctr-go update` for GitHub Release self-updates with platform asset selection and `SHA256SUMS` verification.
- Enabled automatic codex-feishu updates for installed user LaunchAgent services by default.
- Added `/help` version information, styled update checking cards, an interactive update action, and a Feishu completion notice after manual updates.
- Updated install docs to resolve the latest GitHub Release tag dynamically instead of hard-coding an older version.
