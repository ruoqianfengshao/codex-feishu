# v0.6.1

This patch release updates the installation documentation for AI-friendly,
passwordless setup.

## Changes

- Makes the default macOS install path use the release tarball and
  `$HOME/.local/bin` instead of the `.pkg` installer.
- Documents that the user LaunchAgent setup does not require `sudo`.
- Adds architecture detection to the README install snippet for Apple Silicon
  and Intel Macs.
