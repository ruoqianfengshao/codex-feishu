# v0.6.8

This patch release improves Feishu project and thread navigation cards.

## Changes

- Promotes pinned projects and pinned threads to the top of project and chat
  lists.
- Adds pinned labels for pinned projects and pinned threads.
- Applies warmer card colors for project rows, chat rows, and new-thread
  entries.
- Shows `/projects` project updated times as relative time.
