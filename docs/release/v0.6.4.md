# v0.6.4

This release improves Feishu interactive cards and settings workflows.

## Changes

- Reworks `/help` into interactive command cards for workspace and topic usage.
- Replaces `/setting` submenus with a Feishu form and pre-fills model and
  reasoning values from the current Codex config when available.
- Removes language switching from `/status`; language remains configurable from
  `/setting`.
- Refreshes `/status` as a dashboard-style card with KPI sections and a Feishu
  chart component for thread mix statistics.
- Removes the unused `/reply` command from the README command list.
