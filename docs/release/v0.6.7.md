# v0.6.7

This patch release simplifies the Feishu `/status` dashboard.

## Changes

- Removes recent error, delivery queue, and connection details from `/status`.
- Merges thread totals and thread mix into one Threads section.
- Keeps only the Feishu pie chart for thread mix and includes percentages in
  the chart labels.
- Updates README and smoke-test documentation to match the new `/status`
  layout.
