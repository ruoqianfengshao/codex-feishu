# v0.6.6

This release improves Feishu project and chat workflows, Codex desktop
integration, and message delivery reliability.

## Changes

- Lists Codex projects from the current Codex desktop project state so removed
  or archived projects are not shown in `/projects`.
- Shows temporary Codex chats as a separate `/projects` entry and adds a
  projectless "New chat" action.
- Creates temporary chats through the Codex desktop Quick Chat menu action so
  they appear as real Codex projectless chats instead of synthetic project
  folders.
- Keeps project-scoped new threads bound to their selected workspace.
- Improves desktop input retry and app-server fallback behavior when the Codex
  desktop bridge is unavailable.
- Improves Feishu topic routing, image delivery, card updates, and callback
  handling tests.
