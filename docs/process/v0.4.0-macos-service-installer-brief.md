# v0.4.0 macOS service installer brief

`v0.4.0` adds macOS installation and service management without changing the
Telegram runtime contract.

Goals:

- Add `ctr-go service install` with friendly prompts and scriptable flags.
- Keep secrets in `config.env`; LaunchAgent gets only `CTR_GO_CONFIG`.
- Preserve proxy runtime env in `config.env` when needed so the LaunchAgent can
  reach Telegram/Codex through the same network path as the shell.
- Add user-level LaunchAgent lifecycle commands.
- Add a small menu bar tray controller for service actions.
- Publish macOS `.pkg` artifacts in addition to regular CLI archives.

Non-goals:

- Linux systemd and Windows service installers.
- GUI token/settings editor.
- Keychain migration.
- Root/system daemon operation.
